# Portafolio

Portfolio personal construido con FastHTML, HTMX y Alpine.js.

## Stack

- **FastHTML** - Framework web Python para aplicaciones hipermedia
- **HTMX** - Interactividad sin JavaScript mediante atributos HTML
- **Alpine.js** - Animaciones y UI ligera
- **CSS Custom** - Sistema de diseño con variables CSS

## Inicio Rápido

```bash
# Instalar dependencias
pip install -r requirements.txt

# Ejecutar servidor de desarrollo
python main.py

# Abrir http://localhost:5001
```

## Personalización

Edita `data/content.py` para cambiar:
- Información personal y redes sociales
- Experiencia laboral
- Habilidades técnicas
- Proyectos destacados
- Contenido del blog

Edita `static/css/main.css` para modificar colores y estilos.

## Estructura

```
├── main.py           # Entrada y rutas
├── components/       # Componentes reutilizables
├── pages/            # Páginas del blog
├── data/content.py   # Todo el contenido
└── static/           # CSS, JS, imágenes
```
